# CVIA Labs

Coursework notebooks for CVIA. This repo currently contains the Week 1 lab notebooks.

## Contents

- `Week1/`
	- `CVIA_lab_wk1_code1.ipynb`
	- `CVIA_lab_wk1_code2.ipynb`

## Setup

1. Create and activate a Python environment (venv/conda).
2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Open the notebooks in VS Code or Jupyter and run the cells:

```bash
jupyter lab
```

## Notes

- If you add more weeks, mirror the `WeekX/` folder pattern.
- Keep `requirements.txt` updated with any new packages used in notebooks.
